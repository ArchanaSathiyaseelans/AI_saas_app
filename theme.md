# FlowSpace Cozy UI Theme

## Direction
Fresh, cozy, modern, and clean. The app should feel like a calm workspace for thinking, planning, writing, and sketching, while still using enough color to make navigation feel lively.

## Color Palette
- App background: `#f8faf6`, a soft green-tinted white.
- Surface: `#ffffff` for panels and cards.
- Primary text: `#020617` / slate-950.
- Secondary text: `#475569` / slate-600.
- Borders: `#e2e8f0` / slate-200.
- Primary action: teal `#0d9488`.
- Navigation accents: sky, violet, emerald, amber, rose, cyan, lime, fuchsia, and slate.

## Typography
- Use the system sans-serif stack already configured by Next/Tailwind.
- Page titles should be confident but compact.
- Sidebar labels should stay small, readable, and medium weight.
- Group labels should be uppercase with subtle tracking.

## Spacing And Shape
- Use 8px radius for sidebar buttons, panels, controls, and cards.
- Keep sidebar rows compact at about 36px tall.
- Prefer 16-24px page padding and 12-20px panel padding.
- Avoid nested cards; use simple panels and repeated item cards only.

## Interaction Guidelines
- Sidebar must collapse to icon-only mode.
- All icon-only buttons need accessible labels and hover titles.
- Active navigation should be obvious without relying only on color.
- Use colorful Lucide icons to create quick visual scanning.
